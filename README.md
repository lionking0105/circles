# Circles

## Setup

1. Clone the repository

```bash
git clone https://github.com/FlemingVincent/Circles.git
```

2. Navigate to the project directory

```bash
cd Circles
```

3. Install dependencies

```bash
yarn install
```

4. Start the development server

```bash
yarn start
```
